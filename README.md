# SCENERY-preprocessing

### About

This repository contains the **preprocessing code** for data transformation, registration and smoothing.

### Organization Structure

* **SCENERY-preprocessing**: Data cleaning and preparation.
* **SCENERY-models**: Functional model fitting (HR and EDA).
* **SCENERY-results**: Final model outputs and results.
* **SCENERY-FDAmap**: Interactive map visualization of results.
